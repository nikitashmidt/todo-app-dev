Что такое typescript?
TypeScript
- это опциональный язык программирования, который является надмножеством языка JavaScript.
Был разработан в microsoft в 2012 году. Он добавляет статическую типизацию, интерфейсы и другие возможности, которых нет в JavaScript.
Type & Interface
В TypeScript существует два способа определения пользовательских типов данных: интерфейсы (interfaces) и типы (types).
Type -
могут быть использованы для определения простых типов, таких как строки, числа и булевы значения, а также для определения сложных типов, таких как объединение (union) и пересечение (intersection) типов.
interface
- используется для определения формы объекта. Он может содержать только объявления свойств и методов, но не может содержать реализацию.
interface
также может быть расширен другими интерфейсами.
Enums
Enums
(перечисления) в TypeScript позволяют определить набор именованных констант, которые могут быть использованы вместо числовых значений или строк.
Enums
облегчают чтение и понимание кода, а также предотвращают ошибки из-за опечаток в именах констант.
enum Direction {
Up = 'UP' ,
Down = 'DOWN',
Left = 'LEFT'
Right = 'RIGHT'
}
Tuple \ Кортежи
Tuple
- это тип данных, который позволяет определить массив фиксированной длины с определенными типами данных для каждого элемента.
let myTuple: [string, number];
myTuple = ["hello", 123];
Generics
Generics
- это механизм, который позволяет создавать обобщенные типы данных. Обобщенные типы позволяют создавать компоненты, которые могут работать с различными типами данных, не указывая конкретный тип данных заранее.
